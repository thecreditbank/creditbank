Step 1 of 2

Create a Free Account Key

CreditBank uses GitHub to save your data. You need a free "token" (like a password) so the app can save your account and posts. This takes 2 minutes.

1
Create a GitHub Account

Go to github.com/signup and sign up for free. Use any email - you don't need to be a developer.

2
Go to Token Settings

Once signed in, go to github.com/settings/tokens

3
Generate New Token

Click "Generate new token" then "Generate new token (classic)"

4
Fill in the Form

Note: Type "creditbank"
Expiration: Pick "90 days"
Check the box: repo (full control of private repositories)

5
Copy Your Token

Scroll down and click "Generate token". Copy the token that starts with ghp_ and paste it below.

Your token stays in your browser only. We never see it.
